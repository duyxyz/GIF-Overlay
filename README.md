# GIF Overlay - Minimalist GIF & Image Viewer

**GIF Overlay** là một trình xem ảnh GIF và ảnh tĩnh siêu gọn nhẹ dành cho Windows, tập trung vào sự tối giản, hiệu quả và tính thẩm mỹ cao. Ứng dụng cho phép bạn ghim ảnh lên trên các cửa sổ khác với hiệu ứng xuyên thấu độc đáo.

![GIF Overlay Demo](demo1.gif)

## ✨ Tính năng nổi bật

- **Native UI (PyQt5):** Khởi động tức thì, tiêu tốn cực ít tài nguyên hệ thống.
- **Always on Top:** Ghim ảnh luôn nổi trên các ứng dụng khác, lý tưởng để làm mẫu vẽ hoặc trang trí desktop.
- **Translucent Background:** Nền trong suốt hoàn toàn, chỉ hiển thị nội dung ảnh.
- **Hiệu chỉnh linh hoạt:** 
    - Thay đổi độ phân giải (Scale/Width/Height) ngay trong menu.
    - Điều chỉnh độ mờ (Opacity) để nhìn xuyên thấu.
    - Xoay (Rotate) và Lật (Flip) ảnh cực nhanh.
- **High DPI Support:** Hiển thị sắc nét trên mọi màn hình (4K, Laptop scaling).
- **Windows Integration:** Hỗ trợ mở file trực tiếp từ File Explorer (Open With) và đăng ký làm trình xem GIF mặc định.

## 🚀 Cách sử dụng

### Cài đặt
1. Tải file `GIF-Overlay-Setup.exe` từ phần Releases.
2. Chạy bộ cài đặt để đăng ký ứng dụng với hệ thống.
3. Chuột phải vào bất kỳ file `.gif` nào -> **Open With** -> Chọn **GIF Overlay** (có thể tick chọn Always để làm mặc định).

### Điều khiển
- **Di chuyển:** Nhấn giữ chuột trái vào ảnh và kéo để di chuyển.
- **Menu chuột phải:** Chuột phải vào ảnh để mở bảng điều khiển:
    - **Lock Window:** Khóa vị trí và kích thước cửa sổ.
    - **Change GIF...:** Chọn một tệp tin ảnh khác.
    - **Settings:** Điều chỉnh kích thước, độ mờ, xoay và lật ảnh.
    - **Pause / Play:** Tạm dừng hoặc tiếp tục phát GIF.
    - **Quit Application:** Thoát ứng dụng hoàn toàn.

## 🛠 Phát triển

Nếu bạn muốn chạy từ mã nguồn:

1. Cài đặt Python 3.x.
2. Cài đặt thư viện:
   ```bash
   pip install -r requirements.txt
   ```
3. Chạy ứng dụng:
   ```bash
   python "GIF Overlay.py"
   ```
